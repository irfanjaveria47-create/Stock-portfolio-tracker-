# 📈 Stock Portfolio Tracker

## Overview
The Stock Portfolio Tracker is a Python-based application that helps users manage and monitor their stock investments. It allows users to add stocks, track quantities and prices, calculate the total portfolio value, and view profit or loss.

## Features
- Add stocks to your portfolio
- View portfolio details
- Calculate total investment value
- Calculate profit/loss
- Update stock information
- Simple command-line interface
- Easy to customize and expand

## Technologies Used
- Python 3
- CSV (for data storage)
- VS Code (recommended IDE)

## Project Structure
```
StockPortfolioTracker/
│── main.py
│── portfolio.csv
│── README.md
```

## Requirements
- Python 3.8 or higher

## How to Run

1. Clone or download the project.
2. Open the project folder in VS Code.
3. Open the terminal.
4. Run the following command:

```bash
python main.py
```
